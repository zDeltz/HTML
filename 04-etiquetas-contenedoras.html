Etiquetas Contenedor

Las etiquetas div y span en HTML son elementos no semánticos, lo que significa que no aportan ningún significado especifíco sobre su contnido.

Se utilizan principalmente como contenedores generales para agrupar elementos, aplicar estilos CSS o funcionalidades con JS.
